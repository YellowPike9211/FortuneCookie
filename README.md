# fortune cookies

this is a small game i made where you click a fortune cookie and it gives you a random message.

there are 3 types:
- wholesome
- cursed
- funny

they all give different kinds of responses depending on what you pick.

you can play it here:
PUT YOUR ITCH LINK HERE

---

## what this is

i originally made this in python with tkinter but it felt really basic, so i remade it in godot so it actually feels like a game.

now you just click stuff and it responds instead of typing in a console.

---

## how it works (kind of)

each type of fortune is made from a bunch of sentence parts, and it randomly combines them so it’s different every time.

when you click the cookie it switches to a cracked image and then shows the text.

---

## how to run it

just use the itch link above, that’s the easiest way

if you really want the files, you can open it in godot
